# Smart-Posture-Detection-and-Musculoskeletal-Health-Risk-Prediction-
ML-based posture detection system classifying 5 upper-body postures from MediaPipe skeletal landmarks. Compared 4 models (Logistic Regression, SVM, Gradient Boosting, Random Forest), achieving 88.7% accuracy. Deployed as a Streamlit web app with real-time health risk scoring.
